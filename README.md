# Characters Database - Info

## Serviços atualmente disponíveis

### Funcionalidades
* Tratamento de erros personalizados
* Containers de mensagens de retorno de requisições personalizados
* Container de mensagem que engloba todos os outros containers permitindo um tipo de retorno generalizado

### Personagens: /characters
* /save: Serviço responsável por salvá-los na base de dados após uma verificação dos dados
* /obtain: Serviço responsável por retornar uma lista de personagens com base num nome fornecido
* /update: Serviço responsável por atualizar dados de uma personagem na base de dados
